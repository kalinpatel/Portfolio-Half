Tacos el Norte

Resturaunt Website

Tacos el Norte

For this website, I used content from a local resturaunt, "Tacos el Norte," and created a new website. This website was made only to use my design skills, but not sold to the business itself. Firstly, I immediately knew this website should be a single page design. From using the original website from this business, I noticed that some pages would link in a circle. For example, on the menu page there was a Call-To-Action for the catering page, but on the catering page there was a button back to the menu. This was not a very user-friendly design, and by using a single page design with a sticky navigation bar, users could easily scroll up and down the page to find the information they wanted. Secondly, was attractiveness: on the original site, images were blurred and dimmed, and other images were very small. On my website, I used bright colors and a large hero image to let potential customers see the food. In addition, I changed links such as the link to TripAdvisor, and used a colorful button to attrat users to read reviews. Last of all, I included fuctionality: just listing an address is not very helpful to many people who might not know this area, and including COVID-19 information is not relevant in the footer, so I included a Google Maps API so users could find the resturaunt and get directions, and an alert towards the top of the page to let prospective customers know that the resturaunt is still open.

Ecommerce Website

BRAND NAME

This website uses a Gatsby starter template to get up and runnning. For this site, I used Stackbit to automatically configure some settings in my Github Repository to commect this website to Netlify. Using some features Stackbit set up, I set up Netlify CMS for this website. This website was used for a student-run business in which I worked with a team of students to populate website content. While I was able to get the website up and running, and looking good, I needed to use Netlify CMS to allow my fellow peers to edit website content without touching code. The website was initially set up to use SnipCart, but our business eventually moved to a Print-On-Demand service called Redbubble to sell our products.

Why not integrate Redbubble into the store?

Well, Redbubble provides no API's to fetch products and display or sell them on your own website. Instead, they only allow embedding of an iframe containing a list of products. I chose to steer clear of this method because of the lack of control. Firstly, styling is unavailable due to Cross Site Scripting prevention. In addition, link sharing becomes unavailable. When a user wants to share a particular product and copies the link in their browser, any user who opens it will just see a list of products. Unfortunately, Redbubble does not offer an easy way to link to a product using a query and display that itself on my own website. By using a static site generator, I can easily create many pages that can show product information and then send the user to Redbubble for payment processing. While linking to an external website to purchase is not a very user friendly experience, I worked on optimizing my store to be easy to navigate, because having more control over the design is far better than using an iframe with drawbacks.

Event Website

Cobalt Design Conference

I created this website based off of a fictional design conference. Using the Argon Design System (Github Source Code), I created a single page information and registration page for this conference. I opted to use a design system opposed to writing all of the CSS for to purpose of increasing initial development speed. Using an open-source framework to get started allowed me to easily create I website that I could add content to, and then style it to my preference. I chose to use a large hero header to simplify the website, and allow easy access to registration links. The form I used for registration handles responses through a Netlify server, but has been disabled to prevent spam. In this website, I also wanted to use some interactive components. I used tabs heavily, which reduced the page length, so users could click to get more information or scroll past it. In addition, I used the Google Maps API to embed two maps into the site. With the use of JavaScript and the design system I used, small features like tooltips and animations between tabs made the user experience more enjoyable.

What is a Front-end Web Developer?

Web developers take a web design and turn it into code, usually HTML (Hyper Text Markup Language). Web development is usually a long, complicated process, and different types of developers specialize in different areas, which means that large web projects are usually a collaboration between several different developers.

What do I do as a front-end web developer?

I create the visualized parts of a website. In my case, I am also a web designer, as I create the theming and layouts of my website as well as coding them.
