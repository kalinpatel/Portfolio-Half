What is a Front-end Web Developer?

Web developers take a web design and turn it into code, usually HTML (Hyper Text Markup Language). Web development is usually a long, complicated process, and different types of developers specialize in different areas, which means that large web projects are usually a collaboration between several different developers.

What do I do as a front-end web developer?

I create the visualized parts of a website. In my case, I am also a web designer, as I create the theming and layouts of my website as well as coding them. You can see some of my code on Github here, or view the code for this website you are on right now below.
